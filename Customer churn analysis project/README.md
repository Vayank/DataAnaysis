## Overview
This project analyzes customer churn using SQL and Power BI.
- Dataset: Telco Customer Churn
- Tools: Oracle SQL*Plus, Power BI Desktop

## Files
- `TelcoChurnDashboard.pbix` → Interactive Power BI dashboard
- `analysis.sql` → SQL queries for churn rate, contract type analysis, revenue impact
- `telco_churn_clean.csv` → Dataset

## Insights
- Overall churn rate
- Churn by contract type
- Revenue impact of churn
- Tenure group analysis
